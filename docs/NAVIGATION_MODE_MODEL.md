# Navigation, interaction mode, and layout model

The top bar carries recipe identity/status and the primary actions New, Open, Standard/Advanced, Save, Undo/Redo, Compare, and More. Secondary route/library/history/layout/data/import/export/help actions live under More. It stays one row and collapses secondary actions at narrow widths.

Mode controls capability: Standard shows the routine target/precursor/batch/weighing workflow; Advanced adds explicit sites, constraints, overrides, radius descriptors, matrix/rank diagnostics, and the full trace. Mode changes never change scientific state.

Layout controls arrangement: Simple Calculator is the normal two-column workflow; Compact Balance View places the weighing result in the dominant column; Advanced Chemistry activates the information-dense advanced workspace; Route Comparison navigates to the actual comparison workspace. Built-ins are immutable and user layouts are grouped separately.
