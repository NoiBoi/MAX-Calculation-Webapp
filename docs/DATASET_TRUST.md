# Dataset trust and laboratory approval

Scientific-source verification and laboratory approval are independent.

- `source-verified`: primary definition, units, values/qualifiers, and locations were reviewed; usable for exploratory screening.
- `lab-reviewed`: source-verified plus a named local reviewer/date; not approval.
- `lab-approved`: explicit named laboratory approval.
- `provisional`: provenance exists but primary verification is incomplete; aggregates disabled.
- `unverified-import`: imported trust is stripped; aggregates disabled.

No installed radius dataset is currently lab-approved. Source-verified results visibly say so and retain the screening/non-predictive disclaimer. Trust metadata is outside the scientific content digest; changing trust cannot mutate the dataset payload.
